# Creditworthiness assessment

> **Directive:** CCD2 art 18. **Dutch implementation:** Wft 4:34 / 4:34a + AMvB (BGfo arts 113-114).
> Operative text (when transcribed):
> [`extracts/transposition-nl/36924_creditworthiness.md`](../../extracts/transposition-nl/36924_creditworthiness.md).

## What CCD2 requires
- The assessment must be based on information "necessary and proportionate" to the nature, duration, value
  and risks of the credit.
- **Bans** using data obtained from digital social networks and processing **special categories** of
  personal data in the assessment.

## How the Netherlands implements it
- **Art 4:34 Wft** is amended so the assessment rests on information about the consumer's financial
  position that is "noodzakelijk en evenredig"; an exception applies for contract changes that help a
  consumer in payment difficulty without significant credit increases.
- **New art 4:34a Wft** prohibits using digital-social-network data and prohibits processing special
  categories of personal data in the assessment. Art 4:34a(2) creates a legal basis **only** to
  remove/redact special-category data arriving as "bijvangst" (e.g. a payment to a church on a bank
  statement revealing religion), processed separately from the assessment.
- **AMvB (BGfo):** abolishes the **€1,000 verification threshold** (open verification norm, art 113) and
  removes the **€250 BKR-consultation threshold** so BKR is consulted for every agreement (art 114);
  annual re-assessment for revolving credit. — [AMvB](../transposition/implementatiebesluit.md)
